# FirstBank Corporate Identity Guidelines

來源：`sources/FirstBank_Corporate_Identity_Guidelines.pdf`
版本：Abridged Version
研究視角：Design Infrastructure Researcher

## Summary

An abridged FirstBank corporate identity manual covering logo construction, clear zones, product naming, product logo creation, tone of voice, color systems, advertising templates, photography, illustration, merchandise, digital placements, signage, and mobile assets.。建立所有企業識別的核心構造、尺寸、顏色與禁用規則，確保跨市場與跨媒體仍維持同一品牌。

最上層的原則是先建立判斷標準，再進入視覺、語氣或應用規格。recognizability、geometric consistency、contrast、minimum legibility、do-not-redraw governance。logo 系統框架是「core mark -> dynamic shape metric -> logotype alignment -> clear zone -> minimum size -> color system -> misuse prevention」。

執行上，這份 guideline 透過 FirstBank Logo System、SSA Market and Social Media Logo Usage、Product and Service Brand Architecture、Tone of Voice 等章節，把抽象原則落到可操作的資產、版面、語氣、流程或治理規則。大型企業 logo guideline 應把識別元素拆成可計算模組，並給出所有禁止操作。

## FirstBank Logo System

1. **Chapter Purpose**：建立所有企業識別的核心構造、尺寸、顏色與禁用規則，確保跨市場與跨媒體仍維持同一品牌。
2. **Definition**：定義 primary/reversed/mono/secondary logos、Dynamic Shape、內部 spacing、clear zone、minimum size、misuse、color breakdowns、You First lockup。
3. **Relationship**：是後續市場 logo、產品 logo、廣告模板與實體應用的根基。
4. **Design Execution**：作者把 elephant icon、Dynamic Shape 與 logotype descriptor 的位置關係轉成 X-spacing 與 clear zone，讓 logo 組裝可量化。
5. **Principles**：recognizability、geometric consistency、contrast、minimum legibility、do-not-redraw governance。
6. **Frameworks**：logo 系統框架是「core mark -> dynamic shape metric -> logotype alignment -> clear zone -> minimum size -> color system -> misuse prevention」。
7. **Patterns**：以 logo 內部幾何元素作為所有比例量測基準，是可跨尺寸複用的 identity pattern。
8. **Rules**：AI 可檢查：clear zone 必須以 Dynamic Square 高度/矩形量測；horizontal lockup dynamic square 不得低於 4 mm，stacked 不得低於 6 mm；不得 redraw、recreate、translate、recolor、drop shadow、改字體、壓縮、伸展或改變 dynamic shape。
9. **Reusable Knowledge**：大型企業 logo guideline 應把識別元素拆成可計算模組，並給出所有禁止操作。
10. **Abstraction**：`Visual Identity`、`Brand Architecture`、`AI-Checkable Rules`。
11. **Visual Evidence**：logo variants、X-spacing diagram、clear zone、misuse examples 與色彩 breakdown 承擔比例、對照與警示證據。

## SSA Market and Social Media Logo Usage

1. **Chapter Purpose**：管理跨國市場的命名、語言、象徵物與社群 avatar 例外，避免市場擴張破壞主品牌。
2. **Definition**：定義 SSA market、FBNBank UK、DRC elephant omission、Vous d’abord 法文 tagline、social media name appellations。
3. **Relationship**：依賴 logo system，但針對地區、語言與平台提出例外治理。
4. **Design Execution**：作者用市場別明確說明哪些國家保留/移除 elephant、tagline 或 name appellation。
5. **Principles**：localized consistency、platform identification、controlled exception。
6. **Frameworks**：市場識別框架是「market -> language/tagline -> icon presence -> social media identification rule」。
7. **Patterns**：跨市場 guideline 應以例外表管理 localization，而不是讓在地團隊自行詮釋。
8. **Rules**：AI 可檢查：DRC Congo 只用 Dynamic Shape 與 Vous d’abord lockup；English speaking countries 使用 You First and FBNBank lockup except Ghana；社群 profile 可使用 name appellations 以利識別。
9. **Reusable Knowledge**：全球品牌需要把語言翻譯、符號保留與平台 avatar 當成同一組治理問題。
10. **Abstraction**：`Localization`、`Brand Architecture`、`Governance`。
11. **Visual Evidence**：市場 logo 範例與社群示意作為例外判斷證據。

## Product and Service Brand Architecture

1. **Chapter Purpose**：整理歷史產品命名混亂，將既有品牌與未來品牌導入一致的產品識別架構。
2. **Definition**：涵蓋 naming convention、existing product names、future descriptor-first rule、product logo color ratio、divider/header usage。
3. **Relationship**：承接 corporate logo，但處理產品/服務層級的彈性與約束。
4. **Design Execution**：作者保留既有名稱以避免市場成本，卻要求未來命名採 DESCRIPTOR followed by FIRST，並用 70/30 藍金比例建立產品 logo 規則。
5. **Principles**：backward compatibility for live brands、future consistency、endorsed flexibility、ratio-based visual control。
6. **Frameworks**：產品品牌框架是「existing names preserved -> future naming rule -> corporate font for FIRST -> descriptor flexibility -> blue/gold ratio -> divider mechanics」。
7. **Patterns**：以 transition governance 管理品牌債務：不改既有名稱，但統一未來規則與 logo 視覺。
8. **Rules**：AI 可檢查：未來產品名稱應為 DESCRIPTOR + FIRST；FIRST 必須使用 corporate identity font；產品 logo 應採 Blue PMS 295C 與 Gold PMS 124C；agency 應追求約 70/30 balance；divider 不得整組拉伸，應分別調整 bar 長度。
9. **Reusable Knowledge**：品牌架構不只定義理想狀態，也要治理歷史例外與未來命名。
10. **Abstraction**：`Brand Architecture`、`Visual Identity`、`Governance`。
11. **Visual Evidence**：reworked logo examples、product usage、divider ratio guide 與 header guide 是轉換規則的視覺證據。

## Tone of Voice

1. **Chapter Purpose**：讓跨組織溝通具備同一聲音，使品牌不只在 logo 上一致，也在語氣上可辨識。
2. **Definition**：定義 one voice、warm、proactive、personal、flexible、passionate、relevant、collaborative、confident、spirited；區分 ATL brand advertising 與 product/service advertising。
3. **Relationship**：支撐廣告模板、影像與所有 customer communication。
4. **Design Execution**：作者先說明 tone 的商業影響，再列人格屬性與 copywriting 原則。
5. **Principles**：customer-benefit perspective、plain language、warmth、personal connection、anti-jargon。
6. **Frameworks**：語氣框架是「one voice -> brand personality -> communication type -> customer benefit headline -> plain direct copy」。
7. **Patterns**：把語氣拆成內部態度與外部文案規則，可支援文案審核。
8. **Rules**：AI 可檢查：文案應避免 technical jargon 或 corporate speak；headline 應從 customer-benefit perspective 出發；產品文案需 simple and direct；應使用 hello 式親近語氣而非生硬歡迎語。
9. **Reusable Knowledge**：voice guideline 應把抽象 personality 轉成 headline/copy 可檢查行為。
10. **Abstraction**：`Voice and Tone`、`Content Design`、`AI-Checkable Rules`。
11. **Visual Evidence**：此章視覺較少，證據主要是語氣屬性與廣告類型說明。

## Colour Wheel and Endorser

1. **Chapter Purpose**：建立 FirstBank 的主副色與 endorser 應用，使品牌層級與色彩用途明確。
2. **Definition**：定義 Pantone 295C、124C、130C 與 secondary swatches、80/50/30% tint、external applications、FirstBank is an FBN Holdings company endorser。
3. **Relationship**：支援 logo、product architecture、press adverts 與後續所有應用。
4. **Design Execution**：以 color wheel 統整 primary、secondary 與 tint，並把 endorser 放入廣告/企業層級。
5. **Principles**：primary brand dominance、secondary depth、segment application、corporate endorsement。
6. **Frameworks**：色彩框架是「logo colours -> primary swatch -> secondary swatch -> tint percentages -> external application」。
7. **Patterns**：tint ladder 讓同一色彩可在資訊層級與背景深淺中延展。
8. **Rules**：AI 可檢查：Mass Retail image treatments 應使用指定 primary/secondary wheel；外部應用需使用 primary swatch for external applications；endorser 文案應保持固定用語。
9. **Reusable Knowledge**：大型品牌可用 color wheel 將品牌色、區隔色與應用層級整合。
10. **Abstraction**：`Color System`、`Design Tokens`、`Brand Architecture`。
11. **Visual Evidence**：色輪與色票承擔設計 token 證據，endorser 範例承擔品牌層級證據。

## Press Advert and Editorial Templates

1. **Chapter Purpose**：把報紙、廣告與 editorial splash 轉成可重複版面系統，降低代理商自由排版風險。
2. **Definition**：定義 logo/endorser positioning、press advert template、bottom strip template、tabloid single page grid、8mm border、4x2 grid、1mm gutters。
3. **Relationship**：把 logo、divider、tone、color 與 imagery 實作為高頻通路模板。
4. **Design Execution**：作者提供可量測 grid：藍色 8mm border、8 equal rectangles、1mm spacing，並保留特定區域給 logo/headline/subhead/You First。
5. **Principles**：modular layout、protected brand zones、editorial flexibility within grid。
6. **Frameworks**：版面框架是「format -> border -> grid -> reserved zones -> image modules -> text placement」。
7. **Patterns**：將廣告版面轉成 modular grid 是跨媒體可複用的 layout governance。
8. **Rules**：AI 可檢查：tabloid splash 應有 8mm blue border；頁面應分成 4 high x 2 wide 的 8 格；格間距 1mm；top left 與 bottom two rectangles 不應放圖片，應保留給品牌與文字元素。
9. **Reusable Knowledge**：高頻輸出物應以模板和保護區管理，而不是每次重新設計。
10. **Abstraction**：`Layout System`、`Examples`、`AI-Checkable Rules`。
11. **Visual Evidence**：grid diagrams、layout options 與 press templates 是最主要的流程與比例證據。

## Photography and Illustration Style

1. **Chapter Purpose**：定義人物、物件與插畫如何傳達 warm、friendly、bold、clean、simple 的品牌性格。
2. **Definition**：包含 people photography core principles、image choice、FirstBank/FBN Mass Retail photographic style、objects、illustration style。
3. **Relationship**：落實 Tone of Voice 的人格，並支援 advertising、digital、merchandise 的影像一致性。
4. **Design Execution**：作者以 shot focus、background color、subject expression、composition 與業務 segment 反射來控制影像。
5. **Principles**：naturalness、uncluttered composition、aspirational honesty、segment relevance、cohesive tonal quality。
6. **Frameworks**：影像框架是「subject focus -> background/segment color -> expression -> composition -> business reflection」。
7. **Patterns**：用形容詞 cue 加上技術性構圖規則，讓影像審核兼具情緒與可操作性。
8. **Rules**：AI 可檢查：人物照片主焦點應為 face/shoulders/chest；不可像 passport photograph；影像應 natural、uncluttered、real、simple、aspirational、honest、accessible、warm；構圖應 simple, clean, straight talking, not posed。
9. **Reusable Knowledge**：影像系統應把品牌人格翻譯成 casting、crop、background、composition 判準。
10. **Abstraction**：`Photography System`、`Illustration System`、`AI-Checkable Rules`。
11. **Visual Evidence**：攝影樣本、物件示意與插畫風格頁承擔校準與比較證據。

## Printed and Merchandise Applications

1. **Chapter Purpose**：把企業識別落到 cheque book、annual report、flyer、cap、notepad、T-shirt、stationery、banner、vehicle 等實體觸點。
2. **Definition**：定義 printed forms、merchandise dimensions、logo placement、Pantone usage、embroidered text、vehicle/event branding。
3. **Relationship**：將前面所有 identity rules 轉成可生產物件，連接品牌與採購/製造。
4. **Design Execution**：以具體尺寸、顏色、位置與材料說明，降低供應商解讀差異。
5. **Principles**：production readiness、spec-driven consistency、touchpoint coverage。
6. **Frameworks**：應用框架是「asset type -> physical spec -> color/material -> logo/text placement -> example」。
7. **Patterns**：把品牌規範寫成生產規格，是 design system 從畫面延伸到 supply chain 的模式。
8. **Rules**：AI 可檢查：cap color 應為 Pantone 295 C；cap 正面應為 raised embroidered logo；特定品牌語句需維持順序與大小寫；印刷素材需使用指定 Pantone/CMYK。
9. **Reusable Knowledge**：實體應用 guideline 應將品牌規則與製造參數放在一起。
10. **Abstraction**：`Operations`、`Print Production`、`Brand Applications`。
11. **Visual Evidence**：cheque layouts、merchandise mockups、banner/vehicle examples 是供應商與審核者的生產證據。

## Digital Applications

1. **Chapter Purpose**：確保網站、社群、mobile/app、e-flyer、portal banner、email signature 與 HTML template 在尺寸與識別上統一。
2. **Definition**：涵蓋 website dimensions、rendered image area、web banner、social media、mobile/apps、email signature、HTML template。
3. **Relationship**：將 logo、color、tone 與 imagery 帶入數位觸點，並要求尺寸與可視區域正確。
4. **Design Execution**：以 pixel dimensions 與 rendered area 限制關鍵資訊區，避免 responsive 或平台裁切造成品牌損失。
5. **Principles**：digital legibility、platform-aware layout、safe area control、current product representation。
6. **Frameworks**：數位框架是「platform -> canvas size -> safe/rendered area -> brand element placement -> example」。
7. **Patterns**：safe area 與 rendered area 是所有 digital guideline 可複用的基礎模式。
8. **Rules**：AI 可檢查：web banner 應依指定 1200 x 400px canvas 與 871 x 299px rendered area；資訊應置於可渲染範圍內；email signature/HTML template 不得任意改變 logo 與聯絡資訊層級。
9. **Reusable Knowledge**：數位品牌應明確定義畫布與裁切安全區，而不只提供視覺範例。
10. **Abstraction**：`Digital Layout System`、`UI Components`、`AI-Checkable Rules`。
11. **Visual Evidence**：website dimension diagram、banner samples、template pages 提供尺寸與安全區證據。

## Signage and Mobile Truck

1. **Chapter Purpose**：讓實體空間與移動媒體使用同一企業識別邏輯，支援分行導視與戶外曝光。
2. **Definition**：定義 pylon、bulk head、QSP sign、ATM directional signs、department door、in-branch、unit sign、mobile truck。
3. **Relationship**：將 logo、色彩與 typography 應用到環境圖形與交通載具。
4. **Design Execution**：以應用類型分頁展示，讓不同供應商能依物件類別套用。
5. **Principles**：visibility、wayfinding clarity、environmental consistency、scale adaptation。
6. **Frameworks**：signage 框架是「sign type -> viewing context -> logo/color placement -> directional information」。
7. **Patterns**：把 signage 依場景分類，比單一 master template 更能處理尺度差異。
8. **Rules**：AI 可檢查：ATM directional sign 應清楚表達方向；branch signage 必須保持 logo clear zone 與主色；mobile truck 不得使用與品牌無關的大面積圖形。
9. **Reusable Knowledge**：環境識別 guideline 應同時處理辨識、方向與尺度。
10. **Abstraction**：`Environmental Branding`、`Wayfinding`、`Operations`。
11. **Visual Evidence**：signage mockups 與 mobile truck 範例承擔尺度、位置與情境證據。
