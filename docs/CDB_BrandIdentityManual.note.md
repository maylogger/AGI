## Summary

Caribbean Development Bank 的品牌識別手冊，涵蓋基本識別、stationery、advertising、co-branding、corporate communications、special applications 與 multimedia。建立 CDB 品牌一致性的治理正當性，說明任何偏離都會危及品牌完整性。

最上層的原則是先建立判斷標準，再進入視覺、語氣或應用規格。公共金融機構的品牌識別是信任基礎；一致性比創意變化更重要。`brand as shorthand -> consistency requirement -> authorized production -> clarification channel`。

執行上，這份 guideline 透過 Introduction and About These Guidelines、Basic Elements、CDB Logo and Minimum Clear Space、Logo Versions, Colour and Size 等章節，把抽象原則落到可操作的資產、版面、語氣、流程或治理規則。品牌規範可用「governance preamble」把設計規則連到組織風險。

## Introduction and About These Guidelines

1. **Chapter Purpose**：建立 CDB 品牌一致性的治理正當性，說明任何偏離都會危及品牌完整性。
2. **Definition**：定義 brand identity 是對外清楚表達組織屬性的 shorthand，指南供公司內部、設計師、印刷商與授權製作者使用。
3. **Relationship**：這是所有後續規格的政策層，讓 logo、stationery、advertising 與 multimedia 的規範不只是美術要求，而是信任治理。
4. **Design Execution**：用 institution-level language 強調 absolute consistency、strict adherence 與 contact point。
5. **Principles**：公共金融機構的品牌識別是信任基礎；一致性比創意變化更重要。
6. **Frameworks**：`brand as shorthand -> consistency requirement -> authorized production -> clarification channel`。
7. **Patterns**：高信任機構的 guideline 先建立 compliance stance，再給應用細節。
8. **Rules**：AI-checkable criteria：任何輸出應可追溯到本手冊對應章節；若規格不明，應聯絡 CDB Bank Secretary's Unit；未授權方不得自行發起 branded communications。
9. **Reusable Knowledge**：品牌規範可用「governance preamble」把設計規則連到組織風險。
10. **Abstraction**：`Governance`、`Compliance`、`Brand Strategy`。
11. **Visual Evidence**：此章以文字為主，功能是政策與使用者邊界定義。

## Basic Elements

1. **Chapter Purpose**：定義品牌識別最小不可變元素，確保所有後續媒介保持一致。
2. **Definition**：包含 logo 組件、clear space、logo versions、colour specifications、minimum size、unacceptable use、corporate typefaces。
3. **Relationship**：為 stationery、advertising、co-branding、multimedia 提供視覺基礎。
4. **Design Execution**：逐頁以 figure 說明 logo 組成、色彩版本、Pantone/CMYK/RGB、尺寸與錯誤案例。
5. **Principles**：logo 的比例、顏色與元件關係固定；master artwork 是唯一正確來源。
6. **Frameworks**：`logo anatomy -> protected space -> approved versions -> reproduction specs -> misuse controls -> type system`。
7. **Patterns**：先定義 element invariants，再定義媒體應用，是 identity manual 的穩定模式。
8. **Rules**：AI-checkable criteria：logo 只能以一種固定格式使用；logo elements 不得 reposition、recolour、distort、remove、change proportional relationship、apply special effects；signature 必須使用 Arial Bold；print advertising logo diameter 最小 25mm。
9. **Reusable Knowledge**：basic elements chapter 應把 logo 規格轉成 invariants 與可量測條件。
10. **Abstraction**：`Visual Identity`、`Typography System`、`AI-Checkable Rules`。
11. **Visual Evidence**：logo diagram、Pantone table、minimum size figure、unacceptable examples 與 type specimens 是規定證據。

### CDB Logo and Minimum Clear Space

1. **Chapter Purpose**：保護 CDB logo 的可辨識性與權威感。
2. **Definition**：logo 由 medallion、graphic mark/map of Caribbean region 與 signature 組成；clear space 等於 logo type 中 E 的高度。
3. **Relationship**：所有後續頁面放置 logo 時都依此規格。
4. **Design Execution**：用 Figure 1 標註組件，用 Figure 2 定義排除區域。
5. **Principles**：空間與比例是 logo identity 的一部分。
6. **Frameworks**：`component lock-up -> relative measurement -> exclusion zone`。
7. **Patterns**：以字母高度作為 clear-space unit，能跨尺寸縮放。
8. **Rules**：AI-checkable criteria：任何 graphic element、typography 或 pattern 不得進入以 E 高度定義的 clear space；mark 與 signature 的固定關係不得改變。
9. **Reusable Knowledge**：clear-space token 應綁定 logo 內部可見元素，便於人工與機器檢查。
10. **Abstraction**：`Layout System`、`Visual Identity`。
11. **Visual Evidence**：logo anatomy 與 clear-space figure 承擔定義與比例證明。

### Logo Versions, Colour and Size

1. **Chapter Purpose**：確保不同 reproduction 條件下仍維持 CDB 正式樣貌。
2. **Definition**：定義 full colour、flat colour、B&W、reverse colour、reverse B&W 版本，以及 corporate colour palette 與最小尺寸。
3. **Relationship**：為 advertising、co-branding 與 multimedia 的背景與色彩選擇提供規則。
4. **Design Execution**：用 Figure 3/4/5 對應 logo variants、Pantone/RGB/CMYK 與尺寸。
5. **Principles**：在不同媒介中可有版本差異，但顏色與比例不能任意化。
6. **Frameworks**：`background/reproduction need -> approved logo version -> color recipe -> minimum size`。
7. **Patterns**：把 logo version 與 reproduction context 連結是大型機構常見治理模式。
8. **Rules**：AI-checkable criteria：不得使用未授權色彩；印刷廣告最小 logo diameter 為 25mm；若需 colour reproduction，應依 Pantone Inc. 標準配色檢查。
9. **Reusable Knowledge**：色彩表應同時提供專色、印刷與螢幕值。
10. **Abstraction**：`Color System`、`Visual Identity`。
11. **Visual Evidence**：logo version specimens、colour specs 與 size figure 是主要證據。

### Unacceptable Logo Use and Typefaces

1. **Chapter Purpose**：將品牌失真風險轉成清楚禁令，並將字體角色分配到外部與內部溝通。
2. **Definition**：列出 10 項 logo 禁用與 external/internal font families。
3. **Relationship**：補足 Basic Elements 的 enforcement layer，並影響所有媒體模板。
4. **Design Execution**：用 numbered examples 對應錯誤圖，另用 type specimen 展示 Futura、Granjon、Arial、Verdana、Times New Roman。
5. **Principles**：信任品牌不能容許即興變形；字體依用途分層。
6. **Frameworks**：`misuse list -> visual example -> typeface by communication context`。
7. **Patterns**：external communications 使用更具品牌性的字體，internal office tools 使用系統字體。
8. **Rules**：AI-checkable criteria：signature 必須 Arial Bold；collateral 使用 Futura Book/Bold；print advertising body 使用 Granjon Roman/Bold、headers 使用 Futura；internal Word/PowerPoint 使用 Arial/Verdana/Times New Roman；不得 apply special effects 或讓其他 logo dominate CDB logo。
9. **Reusable Knowledge**：typeface policy 應依 production environment 分級。
10. **Abstraction**：`Typography System`、`Governance`。
11. **Visual Evidence**：unacceptable use examples 與 font specimens 承擔警示與選型證據。

## Stationery

1. **Chapter Purpose**：把最私密且高頻的企業溝通媒介標準化，避免個人化 stationery 破壞機構一致性。
2. **Definition**：定義 letterheads、fax sheet、memorandum、business card、compliment slip、envelopes、folder 的紙材、尺寸、logo size、文字位置與 typography。
3. **Relationship**：直接套用 Basic Elements 的 logo、colour、typeface 規則到具體版面。
4. **Design Execution**：每種 stationery 以 figure 提供 precise positioning and measurements，文字列出 paper stock 與 type specs。
5. **Principles**：正式行政溝通中的一致性是品牌信任的日常基礎。
6. **Frameworks**：`document type -> stock -> size -> logo placement/size -> contact info placement -> type specs`。
7. **Patterns**：stationery template specification 可作為 print production checklist。
8. **Rules**：AI-checkable criteria：letterhead logo always top left、30mm；business card logo 25mm 且左側，contact info 右側右對齊；uncoated white stock 為 letterheads/cards/envelopes 的主要要求；contact/address info 使用 Futura BT Medium 與指定 pt/leading/tracking。
9. **Reusable Knowledge**：stationery 規範需包含紙材、物理尺寸、logo 尺寸、字級與對齊，不應只給 mockup。
10. **Abstraction**：`Layout System`、`Operations`。
11. **Visual Evidence**：各 figure 是版面與尺寸證據，功能是生產規格。

## Advertising

1. **Chapter Purpose**：定義 CDB 在不同受眾中的溝通策略，使廣告不只「套 logo」，而能傳達金融穩健與區域發展使命。
2. **Definition**：涵蓋 advertising philosophy、colour palette、brand awareness advertising、A4、posters、promotional items。
3. **Relationship**：沿用 Basic Elements，同時把 messaging strategy 導入 visual layout 與 imagery。
4. **Design Execution**：分 corporate 與 mass audience：corporate 使用強 visual metaphors 與自然主題，mass 使用 testimonials 與成功故事。
5. **Principles**：廣告應避免以貧困臉孔激發罪惡感；應以 solution、prudence、development impact 建立信任。
6. **Frameworks**：`audience segment -> emotional stance -> imagery strategy -> copy strategy -> layout`。
7. **Patterns**：公共發展機構可用不同 audience mode 管理同一品牌下的語氣差異。
8. **Rules**：AI-checkable criteria：corporate advertising imagery 應以 strong photographic nature themes 支援訊息，不以人臉呈現 poverty；mass advertising 可使用 faces，但 copy 應講 achievement/success story；additional campaign material 必須遵守 graphic guidelines。
9. **Reusable Knowledge**：advertising chapter 應把受眾、倫理限制與 visual rhetoric 一起規範。
10. **Abstraction**：`Messaging`、`Photography System`、`Layout System`。
11. **Visual Evidence**：advertising layouts、posters、promotional item examples 承擔示範與語氣比較。

## Co-Branding

1. **Chapter Purpose**：控制 CDB logo 被 partner 使用時的授權、比例與 endorsement 風險。
2. **Definition**：定義 strategic partner logo 使用權、期限、授權流程、official presentation、unacceptable use、proportional sizing、print/stationery/signage/banner/web/promotional rules。
3. **Relationship**：它是 Basic Elements 的外部治理延伸，尤其關係到 CDB 是否被誤認為背書產品或服務。
4. **Design Execution**：用 numbered ground rules、official presentation figures、proportional sizing diagrams 與 application examples。
5. **Principles**：co-branding 是 legal/governance 問題，不只是排版問題。
6. **Frameworks**：`authorization -> permitted purpose -> presentation -> proportion -> placement -> approval`。
7. **Patterns**：partner logo 不得壓過 owner logo，且 owner logo 不應出現在 product endorsement context。
8. **Rules**：AI-checkable criteria：CDB logo remains property of CDB；使用需 pre-authorization 並預留 7 working days；CDB logo should never be bigger than partner logo，理想 1:1 visual space；print/web 上不得出現在 product-oriented communication 或 next to product logo；web 使用應 link to `http://www.caribank.org`。
9. **Reusable Knowledge**：co-branding guideline 應內建授權、比例與 endorsement constraints。
10. **Abstraction**：`Governance`、`Legal and Trademark`、`AI-Checkable Rules`。
11. **Visual Evidence**：official presentation、unacceptable co-branding、proportional sizing 與 application figures 是治理證據。

## Corporate Communications

1. **Chapter Purpose**：讓 press releases、booklets、newsletters、annual reports 等正式出版物保持一致。
2. **Definition**：定義 corporate communications 的 logo placement、date/title/contact info、cover positioning 與版面型態。
3. **Relationship**：介於 stationery 與 advertising 之間，屬於正式資訊發布系統。
4. **Design Execution**：以 figures 展示 press release 與出版物封面、內頁、newsletter、annual report layout。
5. **Principles**：資訊權威來自可預期的版面秩序與低變異 identity placement。
6. **Frameworks**：`communication type -> page size -> logo placement -> text hierarchy -> contact placement`。
7. **Patterns**：把 logo 放置與資訊 hierarchy 模板化，可降低每次出版的設計決策成本。
8. **Rules**：AI-checkable criteria：press release logo top left 30mm，date/title under logo 左對齊，contact bottom right；corporate booklet cover black logo 位於 black title bar 左側 break；annual report full colour logo bottom left。
9. **Reusable Knowledge**：corporate communication templates 應明確定義資訊層級與 logo 位置。
10. **Abstraction**：`Information Design`、`Layout System`。
11. **Visual Evidence**：多頁 mockups 與 measurement callouts 是版面證據；部分 PDF 抽取出現假文亂碼，但不影響主要規格判讀。

## Special Applications

1. **Chapter Purpose**：處理環境、活動與制服等非平面標準媒介，讓品牌在空間與物件上仍一致。
2. **Definition**：包含 corporate signage、banners、uniforms。
3. **Relationship**：把 Basic Elements 與 Advertising/Co-branding 的比例與色彩規則延伸到 physical environment。
4. **Design Execution**：以 entrance signs、directional signs、project signs、banners、uniforms examples 顯示位置與比例。
5. **Principles**：實體環境中的品牌應先服務導引與識別，再服務裝飾。
6. **Frameworks**：`application surface -> logo orientation/size -> colour -> alignment -> informational role`。
7. **Patterns**：signage/banners/uniforms 分別代表 wayfinding、event visibility、personnel identity 三種特殊應用模式。
8. **Rules**：AI-checkable criteria：directional signs must follow logotypes、typography、corporate colours；banners 中 CDB logo should occupy 1/3 of total banner area，vertical banner logo first at top，horizontal banner logo first on left；uniform logo positioned on left chest area。
9. **Reusable Knowledge**：special applications 可轉成「surface-specific identity rules」。
10. **Abstraction**：`Environmental Graphics`、`Operations`、`AI-Checkable Rules`。
11. **Visual Evidence**：signage、banner、uniform figures 承擔情境示範與比例證據。

## Multimedia

1. **Chapter Purpose**：將品牌識別延伸到網站、email、PowerPoint 與 e-newsletter，並處理螢幕媒體的字體、尺寸與可讀性。
2. **Definition**：包含 corporate website、email signature、PowerPoint templates、e-newsletters。
3. **Relationship**：沿用 Basic Elements 的色彩與 typeface，但根據數位媒介改用像素、screen window 與 office software considerations。
4. **Design Execution**：列出 window size、logo size、heading/body font specs，並提供簡報製作 tips。
5. **Principles**：數位品牌治理同時是 identity consistency 與 usability/readability 問題。
6. **Frameworks**：`digital context -> screen size -> logo size -> font hierarchy -> usage tips`。
7. **Patterns**：對 PowerPoint 不只設計模板，也提供 presentation hygiene rules。
8. **Rules**：AI-checkable criteria：website minimum window size 800x600；website logo 34mm；main headings Arial Regular 12pt black or corporate aqua blue；body Arial Regular 10pt black；email signature 應包含 business card 相同資訊；PowerPoint 每張 slide 文字最多 6 lines，landscape format，不混用 screen/print presentations。
9. **Reusable Knowledge**：multimedia guideline 應把品牌 token 與可讀性規則合併。
10. **Abstraction**：`Digital Brand System`、`Accessibility`、`Information Design`。
11. **Visual Evidence**：website/email/PowerPoint/e-newsletter examples 是數位應用證據，功能是模板與操作提醒。
