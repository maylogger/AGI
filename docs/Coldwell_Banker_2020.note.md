# Coldwell Banker Global Luxury Identity Standards

來源：`sources/Coldwell_Banker_2020.pdf`
日期：September 2020
研究視角：Design Infrastructure Researcher

## Summary

Coldwell Banker Global Luxury 的高端房地產識別標準，整合品牌價值、logo/logomark、語氣、色彩、字體、攝影、應用、法務商標與 DBA lockup。先把 luxury 連到 Coldwell Banker 母品牌的服務倫理，避免高端識別只剩黑金視覺符號。

最上層的原則是先建立判斷標準，再進入視覺、語氣或應用規格。luxury 不是誇飾，而是由專業、熱情、創新與卓越服務支撐。brand values -> expression standards -> application governance。

執行上，這份 guideline 透過 Core Values、Logo, Logomark and Voice、Logo、Logomark 等章節，把抽象原則落到可操作的資產、版面、語氣、流程或治理規則。luxury guideline 應把高端感建立在服務證據與文化價值上。

## Core Values

1. **Chapter Purpose**：先把 luxury 連到 Coldwell Banker 母品牌的服務倫理，避免高端識別只剩黑金視覺符號。
2. **Definition**：定義 Home、Awesomeness、Ingenuity、Excellence 四個價值；不負責細部 logo 或版型規格。
3. **Relationship**：為後續 quiet confidence、黑白色彩與高端攝影提供價值根據。
4. **Design Execution**：用使命、歷史、行業創新與服務標準建立 prestige 的來源。
5. **Principles**：luxury 不是誇飾，而是由專業、熱情、創新與卓越服務支撐。
6. **Frameworks**：brand values -> expression standards -> application governance。
7. **Patterns**：高端品牌先定義 service belief，再定義 visual refinement。
8. **Rules**：AI 可檢查：品牌敘事需包含 home joy、expertise、opportunity、exceptional delivery；不得只使用財富或排他性語彙定義 luxury。
9. **Reusable Knowledge**：luxury guideline 應把高端感建立在服務證據與文化價值上。
10. **Abstraction**：`Brand Strategy`、`Messaging`。
11. **Visual Evidence**：本章主要以宣言文字支撐，視覺功能是品牌價值宣示。

## Logo, Logomark and Voice

1. **Chapter Purpose**：建立 luxury extension 的核心識別、seal-like logomark 與克制語氣。
2. **Definition**：涵蓋 stacked/horizontal logo、safety space、logomark、swag/social application 與 quiet confidence copy。
3. **Relationship**：承接核心價值，並供後續色彩、字體、應用與商標章節使用。
4. **Design Execution**：以黑白 logo、wax seal 隱喻、do-not list 與 tone examples 管理識別和語氣。
5. **Principles**：高端品牌以限制、留白與細節建立權威，而非用複雜裝飾增加奢華感。
6. **Frameworks**：mark variants -> constraints -> accent usage -> voice behavior。
7. **Patterns**：把主 logo 與可獨立使用的 logomark 分工，兼顧正式識別與媒介彈性。
8. **Rules**：AI 可檢查：logo 不得小於 circular seal .375 inch；safety distance 至少等於 logomark 高度的 1/2；不得更改色彩、反白內部 star、替換字體、拉伸、傾斜、疊圖、作為文中元素；agent logo 不得大於 Global Luxury logo 且需在 safety distance 外。
9. **Reusable Knowledge**：luxury identity 需要「主標誌」與「細節印章」雙層識別，支援大版面與小 avatar。
10. **Abstraction**：`Visual Identity`、`Voice and Tone`、`AI-Checkable Rules`。
11. **Visual Evidence**：logo versions、social profile、swag mockups 與 copy examples 承擔示範與警示功能。

### Logo

1. **Chapter Purpose**：保護 Global Luxury 與母品牌 rebrand 的連續性。
2. **Definition**：定義 stacked、horizontal、DBA logo variants、尺寸與禁用規則。
3. **Relationship**：是 DBA、stationery、advertising、signage 的基礎。
4. **Design Execution**：用正反版本、黑白版與安全距離控制所有接觸點。
5. **Principles**：luxury mark 必須讓優雅、pedigree 與識別清晰同時存在。
6. **Frameworks**：format fit model：vertical/square materials 用 stacked，horizontal/limited space 用 horizontal。
7. **Patterns**：依版面方向選擇 logo 版本，而非任意縮放。
8. **Rules**：AI 可檢查：logo 必須使用核准黑白版本；黑 box 周圍不得放 copy/graphics；不得用其他配色或字體。
9. **Reusable Knowledge**：logo chapter 應把「版面適配」與「品牌禁用」並列。
10. **Abstraction**：`Visual Identity`、`Legal and Trademark`。
11. **Visual Evidence**：正反 logo、DBA variants、do-not examples 是核心治理證據。

### Logomark

1. **Chapter Purpose**：提供能在社群、swag、accent 上使用的高辨識 seal。
2. **Definition**：logomark 可獨立使用，但必須保持 black/white supplied formats，內部 star 必須為白色。
3. **Relationship**：支援 social media 與 curated luxury swag。
4. **Design Execution**：以 profile image、item placement、light/dark examples 展示彈性。
5. **Principles**：secondary mark 應增強品牌細節感，但不能取代完整 trademark。
6. **Frameworks**：primary identifier vs accent identifier。
7. **Patterns**：seal-like monogram 可作為 luxury brand 的 social avatar pattern。
8. **Rules**：AI 可檢查：logomark 不得 negative 使用；star 不得改成非白色；例外僅 blind emboss。
9. **Reusable Knowledge**：對數位與實體小物件，獨立 mark 是必要識別資產。
10. **Abstraction**：`Iconography System`、`Visual Identity`。
11. **Visual Evidence**：社群頁面與 swag mockups 說明 mark 的情境可用性。

### Voice

1. **Chapter Purpose**：避免 luxury 文案過度炫耀，建立從容、清楚、禮貌的高端語氣。
2. **Definition**：定義 thoughtful、informative、clear、courteous、quiet confidence。
3. **Relationship**：影響廣告、email、social 與 listing copy。
4. **Design Execution**：以三種 tone examples 展示風景描述、專業說明與短句標語。
5. **Principles**：高端溝通要尊重客戶時間與責任，透過 grace 而非壓迫感建立權威。
6. **Frameworks**：thoughtful imagery copy + informative credibility copy + clear headline。
7. **Patterns**：用不同文案長度承擔不同功能：氛圍、證據、記憶。
8. **Rules**：AI 可檢查：copy 應清楚、禮貌、資訊充分；不得過度誇張、粗俗、硬銷或不必要地複雜。
9. **Reusable Knowledge**：luxury voice 可轉為「低噪音、高證據、強節制」內容準則。
10. **Abstraction**：`Voice and Tone`、`Content Design`。
11. **Visual Evidence**：文案樣例搭配留白呈現 quiet confidence 的節奏。

## Color and Typography

1. **Chapter Purpose**：用精煉色彩與字體階層把 luxury 轉為可視語言。
2. **Definition**：涵蓋黑白灰主色、Camel Hair、Aegean、digital palette、UI usage、serif/sans/geometric type roles。
3. **Relationship**：為 applications 的所有版型提供 token 與文字風格。
4. **Design Execution**：黑白灰創造力量與簡潔，照片承擔主要色彩；字體負責在 refined、bold、conversational 之間調節。
5. **Principles**：高端色彩不靠多色，而靠控制與留白；字體選擇必須回應 message mood。
6. **Frameworks**：neutral surface -> photographic color -> sparse specialty accents；serif elegance + sans clarity + geometric emphasis。
7. **Patterns**：讓色彩退居背景，把物件與人物變成真正的色彩來源。
8. **Rules**：AI 可檢查：UI surface 應為 black 或 white；specialty colors 只能 sparingly 使用；black 為 typography go-to；Geometos 用於 headlines/subheads/charts/standout phrases。
9. **Reusable Knowledge**：高端系統可將 palette 壓縮，讓攝影與材質承擔情緒。
10. **Abstraction**：`Color System`、`Typography System`、`Design Tokens`。
11. **Visual Evidence**：色票、數位色板與 type specimens 是 token 證據。

### Color Palette

1. **Chapter Purpose**：建立 luxury 的克制視覺基底。
2. **Definition**：Noir、Blanc、Ebony、Pewter、Fog、Aegean、Camel Hair 與數位 primary/secondary/blue variants。
3. **Relationship**：直接影響 UI、廣告、stationery 與 signage。
4. **Design Execution**：明確說明 color should be introduced through photography and illustrations。
5. **Principles**：品牌色越克制，越需要影像品質與留白支撐情感。
6. **Frameworks**：primary surfaces + specialty functional accents。
7. **Patterns**：digital palette 另列，避免 print palette 直接誤用到 UI。
8. **Rules**：AI 可檢查：primary buttons 使用 Material contained primary；secondary buttons 使用 outline primary 或 text only；accent 遵守 Material primary/secondary rules。
9. **Reusable Knowledge**：print 與 digital color systems 應有橋接但不可混為一談。
10. **Abstraction**：`Color System`、`UI Tokens`。
11. **Visual Evidence**：色票與 UI rule text 承擔設計 token 證據。

### Typography

1. **Chapter Purpose**：讓同一品牌能在優雅、力量、清晰與親近間調節聲音。
2. **Definition**：Adobe Garamond Pro、Helvetica Neue LT Std、Geometos Neue；digital alternatives 為 Helvetica/Garamond、email 為 Arial/Times New Roman。
3. **Relationship**：服務 voice 與 application templates。
4. **Design Execution**：提供不同字體處理如何傳達 sophisticated、clear、conversational、bold、elegant。
5. **Principles**：字體不是單一規格，而是 mood selector。
6. **Frameworks**：font family -> emotional role -> channel fallback。
7. **Patterns**：同時規定 brand fonts 與 channel-safe substitutes。
8. **Rules**：AI 可檢查：recruiting powerful messages 可用 all caps/bold Geometos；serif italic 可用於 smart conversational；Helvetica Neue lower/upper lowercase headline 可呈現 personal inviting。
9. **Reusable Knowledge**：typography examples 應說明「何時用哪種字體語氣」。
10. **Abstraction**：`Typography System`、`Voice and Tone`。
11. **Visual Evidence**：字體樣張與語氣詞直接連結視覺感受與文案角色。

## Photography

1. **Chapter Purpose**：用物業影像建立 luxury 的可信感、情境感與感官細節。
2. **Definition**：分為 exterior、interior、details。
3. **Relationship**：支撐 advertising、social、collateral 的主要情緒來源。
4. **Design Execution**：exteriors 強調 dramatic angles and beautiful lighting；interiors 強調 atmospheric、dramatic、welcoming；details 強調 textures、colors、whimsy。
5. **Principles**：luxury 由空間、光線、細節與生活想像共同構成。
6. **Frameworks**：architecture context -> lived interior -> sensory detail。
7. **Patterns**：三層攝影系統：大場景、室內體驗、細節證據。
8. **Rules**：AI 可檢查：exterior 需突顯建築與環境；interior 需呈現 unique environments 且具歡迎感；detail shots 需提供 texture/color/spark，不應只是填充圖。
9. **Reusable Knowledge**：高端房地產攝影需同時有 scale、atmosphere、material proof。
10. **Abstraction**：`Photography System`、`Visual Language`。
11. **Visual Evidence**：整頁照片是主要視覺證據，承擔品質標準與情境示範。

## Applications

1. **Chapter Purpose**：把識別系統落到代理人與辦公室每天會用的素材，降低自由發揮造成的不一致。
2. **Definition**：涵蓋 stationery、advertising、collateral、social、email、event materials、signage。
3. **Relationship**：依賴 logo、type、color、photography 與 trademark copy。
4. **Design Execution**：以大量 mockups 與尺寸/字體/法律字提供直接可複製範本。
5. **Principles**：成熟 franchise 品牌需要模板化，讓 local office 自主製作仍保持全域一致。
6. **Frameworks**：asset type -> template options -> legal copy -> production source。
7. **Patterns**：同一版型提供 option 1/2，兼顧一致性與微彈性。
8. **Rules**：AI 可檢查：所有 external materials 需保留合法 disclaimer；listing ad 需包含 property info、represented by、search MLS 或 URL；stationery 需包含 each office independently owned and operated。
9. **Reusable Knowledge**：application chapter 應將法律 copy 與視覺模板綁定，避免規定遺漏。
10. **Abstraction**：`Templates`、`Operations`、`Compliance`。
11. **Visual Evidence**：名片、信紙、廣告、postcard、signage mockups 是成品比對證據。

### Stationery

1. **Chapter Purpose**：統一高端房仲個人與辦公室的正式接觸點。
2. **Definition**：business card、letterhead、note card、envelope、housing folder。
3. **Relationship**：承接 logo/type/color，為代理人日常文件提供標準。
4. **Design Execution**：黑底、白底、黑箔、embossed 等選項建立 tactile luxury。
5. **Principles**：高端品牌的紙本文具需讓材質與細節支撐信任。
6. **Frameworks**：front/back options + contact hierarchy + legal ownership line。
7. **Patterns**：模板化個人化欄位。
8. **Rules**：AI 可檢查：姓名、職稱、地址、電話、email、URL、license、independently owned line 需依模板存在；不得任意改變字距風格。
9. **Reusable Knowledge**：stationery 是 personal brand 與 corporate identity 的交會模組。
10. **Abstraction**：`Stationery System`、`Templates`。
11. **Visual Evidence**：多頁 stationery mockups 是生產規格證據。

### Advertising and Collateral

1. **Chapter Purpose**：提供可直接套用的 luxury property marketing 版型。
2. **Definition**：branding ads、seller ads、listing ads、magazine spread、postcards。
3. **Relationship**：把 voice 與 photography 放進銷售素材。
4. **Design Execution**：用 headline、property data、agent info、URL 與 legal copy 組合。
5. **Principles**：高價 listing 需要精簡訊息、強照片、清楚代表資訊與免責聲明。
6. **Frameworks**：brand message -> property facts -> agent attribution -> legal protection。
7. **Patterns**：同一房源資料可轉譯到單頁、跨頁與 postcard。
8. **Rules**：AI 可檢查：價格、bed/bath、MLS/search line、COLDWELLBANKERLUXURY.COM、property disclaimer 不可缺；所有資訊需避免未驗證承諾。
9. **Reusable Knowledge**：real estate ads guideline 應把資訊架構和法務風險視為同一系統。
10. **Abstraction**：`Advertising Templates`、`Compliance`。
11. **Visual Evidence**：listing grids 與 magazine spreads 承擔版面密度示範。

### Social, Email and Event Materials

1. **Chapter Purpose**：讓數位與活動觸點維持 luxury 語氣。
2. **Definition**：Facebook/Instagram posts、HTML eblast、pull-up banner、table skirt、tent cards。
3. **Relationship**：延伸 logo/logomark、voice、type 到短格式與活動環境。
4. **Design Execution**：提供 600px email layout、40px margins、80px logomark 等規格。
5. **Principles**：短格式仍需有節制、可讀與法務完整性。
6. **Frameworks**：channel dimensions -> typographic fallback -> disclaimer inclusion。
7. **Patterns**：數位素材規範需給出 pixel-level constraints。
8. **Rules**：AI 可檢查：email 使用 web safe fonts Times Italic/Arial Regular；layout 基於 600px x 900px；margins 40px；活動物料需保留 Each office is independently owned and operated。
9. **Reusable Knowledge**：活動與 email 需要同時管理尺寸、字體 fallback 與法律文字。
10. **Abstraction**：`Digital Templates`、`Event Branding`。
11. **Visual Evidence**：email mockup、banner/table/tent card 例圖展示媒介轉譯。

### Signage

1. **Chapter Purpose**：讓 yard signs 在高度本地化的環境中仍保有統一結構。
2. **Definition**：yard signs、directional signs、sign riders 與 contact information variants。
3. **Relationship**：連結 DBA、logo、local regulations 與 real-world visibility。
4. **Design Execution**：以 top two-thirds for company branding、bottom third for contact info 建立固定資訊架構。
5. **Principles**：標示必須遠距可辨、資訊清楚且符合地方房地產規範。
6. **Frameworks**：brand zone + contact zone + local compliance responsibility。
7. **Patterns**：用固定比例區塊管理 signage 資訊優先級。
8. **Rules**：AI 可檢查：上方 2/3 保留公司品牌，下方 1/3 為聯絡資訊；variation 必須遵守 black/platinum design；使用者需確認地方 real estate regulations。
9. **Reusable Knowledge**：環境標示需同時定義比例、資訊層級與規定責任。
10. **Abstraction**：`Environmental Branding`、`Compliance`。
11. **Visual Evidence**：大量 yard sign variants 是現場可見性與版型治理證據。

## Identity Requirements

1. **Chapter Purpose**：保護 trademark、service mark、DBA 與 legal disclaimer，降低 franchise 與跨區使用風險。
2. **Definition**：包含 trademark overview、disclaimer、mark in copy、notation definitions。
3. **Relationship**：為所有 applications 提供規定底線，並銜接 DBA lockup。
4. **Design Execution**：用 REQUIRED 與 DO NOT 條款把法務語言轉成操作限制。
5. **Principles**：品牌識別與法律權利不可分離；local DBA 自由度必須受控。
6. **Frameworks**：mark ownership -> approved DBA -> required disclaimer -> copy notation。
7. **Patterns**：legal rule chapter 應提供 examples of inappropriate trade names。
8. **Rules**：AI 可檢查：disclaimer 必須完整不可改；Coldwell Banker Global Luxury® 首次出現需加 ®；不得縮寫為 CB Global Luxury、CBGL 或複數化；COLDWELLBANKERLUXURY.COM standalone 需 uppercase/no spaces。
9. **Reusable Knowledge**：規定規範應以可 lint 的名稱、符號與 copy rule 實作。
10. **Abstraction**：`Legal and Trademark`、`Compliance`、`AI-Checkable Rules`。
11. **Visual Evidence**：此章以文字法規為主，功能是規定證據。

## Logo with DBA

1. **Chapter Purpose**：解決 franchise/local office 名稱與 Global Luxury mark 的組合問題。
2. **Definition**：涵蓋 vertical、horizontal、stacked、single-line、double-line、long names 的 DBA lockup。
3. **Relationship**：依賴 identity requirements 的 pre-approved DBA 與 logo variants。
4. **Design Execution**：用 W、X、Y、Z、V 等測量單位定義字寬、高度、leading、padding、box alignment。
5. **Principles**：DBA 是品牌架構的一部分，不是 logo 下方可隨意放置的文字。
6. **Frameworks**：orientation -> character count -> line count -> scale/alignment rules。
7. **Patterns**：以 character threshold 選擇 lockup template。
8. **Rules**：AI 可檢查：vertical single-line DBA <=12 characters；vertical stacked single-line <=10 characters；horizontal single-line <=10 characters；水平 stacked single-line <=12 characters；double-line preferred line length 約 12-15 或 13-15 characters；long names 需 scale to guide box 並依規則對齊。
9. **Reusable Knowledge**：DBA lockup 可抽象成 responsive logo composition system。
10. **Abstraction**：`Logo System`、`Brand Architecture`、`AI-Checkable Rules`。
11. **Visual Evidence**：測量圖與 guide boxes 是比例、流程與版型治理證據。

## Closing

1. **Chapter Purpose**：以品牌標語與法律文字收束，提醒所有素材仍受 copyright 與 trademark 控制。
2. **Definition**：包含 Live a life of LUXURY 與 final legal copy。
3. **Relationship**：回扣 voice 與 identity requirements。
4. **Design Execution**：以簡潔收尾保留高端節奏。
5. **Principles**：品牌文件結尾也應是規定與語氣的一部分。
6. **Frameworks**：brand line + legal close。
7. **Patterns**：用一句品牌語作為記憶錨點。
8. **Rules**：AI 可檢查：使用 closing legal copy 時不得刪改 trademark、ownership 與 Fair Housing/Equal Opportunity statement。
9. **Reusable Knowledge**：closing page 可同時承擔品牌記憶與法律保護。
10. **Abstraction**：`Governance`、`Legal and Trademark`。
11. **Visual Evidence**：以標語和法律文字為主，視覺證據較少。
